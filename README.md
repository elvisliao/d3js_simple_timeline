## 介紹
使用ES5 D3.JS實現HTML5播放器的時間軸

## 功能
- 時間範圍縮放：可依照不同級距顯示。
- 畫面切換：點擊式切換畫面加上過場動畫，不同於一般的slider。
- 指標拖曳：實現播放時間點位
- 回到live：將指標/畫面帶回最新的時間點。
